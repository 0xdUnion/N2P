# 路由

路由可以让单页包含多个页面，通过`#/`来识别

## 最佳实践

路由的配置文件：`src/router/index.js`，简化来讲这样就可以：
~~~json
{
    path: '/home',
    component: () => import('../views/HomeView.vue'),
},
{
    path: '/about',
    component: () => import('../views/AboutView.vue'),
}
~~~

完全不必在开头导入

# Ciallo

接下来我们来创建一个页面

首先创建视图：`src/views/HelloView.vue`

~~~vue
<template>
    Ciallo～(∠・ω< )⌒☆
</template>
~~~

然后在`src/router/index.js`中加入

~~~json
{
    path: '/hello',//路径/hello
    component: () => import('../views/HelloView.vue'),//展示HelloView.vue视图
}
~~~

接下来访问`http://localhost:5173/hello`，你应该会看到某句话～(∠・ω< )⌒☆
