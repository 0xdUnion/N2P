# 方向选择

> 欢迎👋，接下来可以选择自己最适合的学习方向。开发是一个团队协作的过程，因此每个成员专精一个方向即可

<img src="/images/stickers/Tachibana_Sheri_AgADFhoAAp2ieFc.gif" width="200px"/>


## 前端Vue

国内广泛应用的前端框架，适合习惯类html语法的同学学习  
中文生态成熟，传闻ai吃的vue代码远超react

[https://cn.vuejs.org/](https://cn.vuejs.org/)

## 前端React

全球知名前端框架，大厂爱用，ui库丰富  
语法与原生html差异较大

[https://zh-hans.react.dev/](https://zh-hans.react.dev/ )

**注意** React共学由航电链协提供，可能需要一些额外申请

## 后端Solidity

以太坊合约开发语言  
由于搭配hardhat,~~使用js/ts是逃不掉的！~~

[https://learnblockchain.cn/docs/solidity/](https://learnblockchain.cn/docs/solidity/)

## 全栈

选择一个前端框架配合Solidity，一人包办前后端

## Rust

!> Rust难度较大，推荐有编程基础的同学选择

一门近几年很热的语言，用于各种领域的开发  
在区块链还能应用于底层建设

**注意** Rust共学由航电链协提供，可能需要一些额外申请


[https://rust-lang.org/](https://rust-lang.org/)
